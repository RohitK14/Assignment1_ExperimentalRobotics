Detailed Description

This class defines the state of the state machine corresponding to the robot randomly moving in the 2D plane.

It is a inheritance from smach and the state is added to the smach_state. In this state the robot moves around randomly by subscribing to the rostopic /moveToPose. It has a function execute() providing the intended behavior.
